# Health App - Backend

Backend приложения для управления здоровьем с интеграцией Telegram WebApp.

## Технологии

- Node.js + Express
- PostgreSQL
- TypeScript
- Telegram WebApp Authentication (HMAC-SHA256)

## Быстрый старт

```bash
cd backend
npm install
cp .env.example .env
# Настройте переменные окружения в .env
npm run migrate
npm run dev
```
